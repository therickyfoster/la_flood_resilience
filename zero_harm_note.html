Zero-Harm Principles

Ethical guidance for emergency response that prioritizes safety, consent, and transparency.

Core Commitment

This application is designed exclusively for public safety, emergency preparedness, and community resilience. It contains no features that could be repurposed for surveillance, enforcement, or coercion.

This application does not collect, transmit, or store any user data externally. There is:

  • No user tracking or analytics sent to external servers
  • No location data collection beyond what user voluntarily provides
  • No cookies used for tracking purposes
  • No integration with advertising or data broker networks
  • No hidden telemetry or background data transmission

All user preferences and application state are stored locally on your device using IndexedDB:

  • Language preferences stored locally
  • Theme settings stored locally
  • HUD metrics stored locally
  • Search history never transmitted
  • User can clear all data at any time via browser settings

Every interaction with this application respects user autonomy:

  • No pop-ups, modals requiring action, or dark patterns
  • Information provided without requiring registration or identification
  • Offline functionality ensures access without connectivity requirements
